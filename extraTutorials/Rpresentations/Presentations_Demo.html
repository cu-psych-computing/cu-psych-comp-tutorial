Why should I make my presentation in R?

  • No more copying and pasting graphs into powerpoint or keynote.

  • Great for data presentations and lab meetings where you need to show lots of graphs.

  • Great for tutorials or labs where you want to show code and output.

  • Maybe dont use this when you really really care about how your slides are laid out.

Arranging your text

  1. Hashtags can be used to add headers of various sizes

  2. Start lines with numerals for a numbered list, or use (*) for unnumbered lines

  3. End a line with two spaces to start a new paragraph.

Block quotes start with (>)

  • See code for how to do:
  • italics
  • bold
  • superscripts2
  • strikethrough
  • link

knitr::include_graphics("Evie.jpg")
